- Sublime Package Manager

- Emmet
    + In sublime: `cmd/win + shift + p`
    + Search `install`
    + Choose `Package Control: Install Package`
    + Search `Emmet` (ex-zen coding for sublime)

    + In sublime: `cmd/win + shift + p`

    + Choose `Package Control: Install Package`

    + Install

    + In sublime: `cmd/win + shift + p`
    + Search `install`
